# week2 linked list
是由node(節點)與pointer(指向)所組成

node(節點)可以想成箱子(就是在記憶體所佔的位置)

pointer(指向)就是箱子跟箱子之間的箭頭指向

linked list的好處就是在記憶體中不需要一個連續的空間來儲存，可以利用零散的位置來儲存資料

然而array卻相反，但是array在尋找資料的時候會比較簡單，而且array在搬運的時候會很麻煩，因為要整組一起
